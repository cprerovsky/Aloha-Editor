Building Aloha Editor

Use the build script build.sh to build Aloha Editor. You don't need to add any parameters. The optimized and minified aloha.min.js file can be found in the build directory after building.

NOTE: Executing the build script will wipe all contents of the build directory.

Build options

The build script currently takes two options

  • --simple build with simple optimizations. Quicker, but less reduction in size. Not suggested.
  • --source-map will also create a working source map and place all the source files in the build folder.
